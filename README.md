# KotekTools-PC
KoTeKTools on PC

KoTeKTools:

Мощный инструмент созданный для упрощения работ со сборками.

Функционал:
**Конвертации**  
• Конвертация PNG⇄BTX  
• Конвертация ANI⇄IFP  
• Конвертация MOD→DFF  
• Конвертация BPC→BPCMETA  
• Конвертация BPC→ZIP  
• Конвертация TXD→ZIP  

**Копирование**
• Копирование БилдБордов  
• Копирование Логотипов  
• Копирование Листвы  

**Атмосфера**  
• Изменение неба  
• Изменение окружения  

**Покраска**  
• Покраска Кнопок  
• Покраска HUD-элементов  
• Покраска изображений  

**Дополнительно**  
• Изменение крови  
• Нарезание карты  

**Текстуирование**  
• Работа с текстурами(YTD)  

**3D Просмотр**  
• 3D Модель(DFF)
